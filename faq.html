TUMBLETURN FAQ

VIEW TIMES:
Select the swimmer from the list, grouped by club, and the swim distance. The next field is 'Entries' which is the number of recorded times to list. This is a sorted list with the fastest (PB) first so entering '1' returns the PB and entering '10' returns the top ten times. Then select the pool length and finally the stroke button will take you to the results page. Selecting all PBs will list them all for the pool length selected. The Hy-Tek site requires you to accept their cookies so the first time you access the site you will need to select the green coloured 'Accept' button and then resubmit the search from tumbleturn.

SEARCH FOR SWIMMERS:
Enter a surname, or part of, and a club name to search within that club for matching swimmers. The search button will take you to the swimmer's Hy-Tek page. Leaving the search field blank returns all the swimmers from that club.

GB RANKINGS:
This page is used to view country-wide rankings and recorded times on the swimmingresults.org website, for assessing eligibility for national level meets. Search by family name or go straight to the rankings selection page to select age groups and events.

25M TO 50M CONVERSION:
ASA table conversion using the Sportsystems algorithm. Please note that competition entry times may be based on different calculations so check with the meet organiser before submitting times.

QT/CT TIMES:
Links are provided to the current list of qualifying and consideration times for major meets.

PB CHECK:
Compared your PB against the current list of qualifying and consideration times for major meets.
